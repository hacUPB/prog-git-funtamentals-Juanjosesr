# crear un repositorio remoto  
  
Ingresa a la página de github y logeate o registrate, creas un nuevo repositorio y copias el link que sea específicamente de ese nnuevo repositorio tuyo que creaste (link único), ahora en git bash debes tener un repositorio ya creado y abierto, ejecutas el comando  
- git remote add origin []   
con tu link en ves del [], ahora para subir tus cambios locales al repositorio remoto, ejecutas el comando  
- git push -u origin main  
Y para traer los cambios hechos en remoto ejecutas el comando  
- git pull origin main
