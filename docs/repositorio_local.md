# crear repositorio local 
para crear un repositorio primero te tienes que dirigir a la carpeta en donde deseas dejar guardado tus proyectos e información trabajada, siendo como por ejemplo en mí caso la carpeta documents (documentos), ya estando acá con el comando  
- mkdir  
creamos una carpeta con el nombre que querramos, como por ejemplo Programacion_2026 y ya después de creada ingresamos a la misma y creamos una nueva carpeta que será  la que usaremos como repositorio, como por ejemplo Repo1, ya después de creada ingresamos a la misma y digitamos el código  
- git init  
ya tendriamos nuestro repositorio listo para trabajar ;)
