El archivo .gitignore es un archivo de texto que le dice a Git qué archivos y carpetas debe ignorar en un proyecto, evitando que se incluyan en el control de versiones, como archivos de configuración local, temporales o de compilación; se coloca en la raíz del proyecto y contiene patrones de nombres de archivos/carpetas, usando * como comodín y # para comentarios, y también puede haber una versión global para ignorar archivos en todos los repositoris
